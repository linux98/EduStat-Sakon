1. บทสรุปผู้บริหารและข้อมูลเทคนิคทั่วไป

ระบบจัดเก็บสถิติเชิงลึกจังหวัดสกลนคร (EduData Sakon) ได้รับการพัฒนาขึ้นโดยมีเป้าหมายเพื่อปฏิรูประบบการเก็บรวบรวมข้อมูลสถิติทางการศึกษา จากเดิมที่เป็นการดำเนินการผ่านไฟล์เอกสารที่ไม่เป็นระบบ ให้กลายเป็นการส่งรายงานข้อมูลแบบไดนามิกผ่านเว็บคลาวด์แอพพลิเคชันที่รวดเร็ว ถูกต้อง และปลอดภัย ระบบได้รับการพัฒนาด้วยสถาปัตยกรรม Google Apps Script (GAS) ซึ่งช่วยลดค่าใช้จ่ายด้านเซิร์ฟเวอร์ลงเหลือ 0 บาท (Serverless Architecture) ในขณะที่สามารถผสานการทำงานกับ Gemini AI API เพื่อช่วยการวิเคราะห์เชิงลึกได้อย่างมีประสิทธิภาพ

2. รายละเอียดสถาปัตยกรรมฐานข้อมูลเชิงสัมพันธ์
2.1 โครงสร้างชีตฐานข้อมูล (Database Schema)

ระบบใช้ Google Sheets จำลองแบบ Relational DB โดยมี Referential Integrity ผ่าน `AgencyId` หรือ `FormId` และดึงค่าผ่านแอดเดรสแคชที่เสถียร แยกออกเป็น 6 ชีตหลัก:

ชื่อตารางชีต โครงสร้างคอลัมน์ (Fields) คีย์เชื่อมโยง หน้าที่การเก็บข้อมูล
Users Username, PasswordHash, Role, AgencyId, Email, Name PK: Username บัญชีเข้าใช้งานจำแนกสิทธิ์ Super Admin, Admin, Agency
Settings SettingKey, SettingValue, Description, LastUpdated PK: SettingKey การตั้งค่าระบบและ Deadline กำหนดส่งรายงาน
FormTemplates FormId, Title, FieldsJson, Status, CreatedBy, CreatedAt PK: FormId โครงสร้างคำถามฟิลด์แบบฟอร์มที่แอดมินสร้างขึ้นแบบ Dynamic
Data DataId, FormId, AgencyId, SubmissionDataJson, Status, SubmittedAt FK: FormId, AgencyId ข้อมูลดิบที่สถาบันนำเข้าและส่งรายงาน รวมถึงแบบร่าง (Draft)
AuditLogs LogId, Timestamp, Username, ActionType, Detail, IpAddress FK: Username บันทึกกิจกรรมความปลอดภัยเพื่อใช้ในการทวนสอบ (Audit Trail)
GisCoords AgencyId, Lat, Lng, StudentCount, TeacherCount, NeedsTeacher PK/FK: AgencyId พิกัดแผนที่ GIS จำนวนครู-นักเรียนของโรงเรียนสังกัด เพื่อเรนเดอร์ GIS Map
2.2 รายละเอียดฟังก์ชันระบบเด่นส่งมอบจ้าง (The "WOW" Feature Suite)
  • ระบบแผนที่อินเตอร์แอคทีฟจับคู่ตารางคะแนน (GIS Map & Leaderboard Sync): แผนที่แสดงเลเยอร์ความหนาแน่นของผู้เรียน สลับเฉดสีครูค้างส่ง และการลิงก์ความก้าวหน้าโรงเรียน (Leaderboard) เมื่อคลิกที่อันดับจะดึงพิกัดแผนที่ซูมเข้าทันทีและอัปเดตแบบไร้การกระตุก (Silent Reload)
  • ผู้ช่วยส่วนตัววิเคราะห์ข้อมูลระดับอัจฉริยะ (Conversational BI Assistant): การรับค่าผ่านกล่องอินพุตคำถามภาษาไทย เช่น "ขอชาร์ตสัดส่วนครูต่อนักเรียนรายอำเภอเป็น Radar Chart" ส่งเข้า Gemini AI แปลงเป็นโครงสร้าง JSON อัตโนมัติและดึงข้อมูลจากฐานข้อมูลมาวาดชาร์ตกราฟบนจอแอดมินใน 3 วินาที
  • ระบบเตือนทวงงานแบบกำหนดบุคคลด้วย AI (AI Auto-Dunning Email): ระบบแสกนตรวจสอบโรงเรียนที่เลยกำหนดดีดไลน์อัตโนมัติ สรุปจำนวนวันค้างส่ง แล้วสั่งการให้ AI ร่างหนังสือทางการที่มีเนื้อหาดุดันแต่สุภาพส่งตรงผ่านอีเมลผู้บริหารโรงเรียนนั้นทันทีพร้อมลงบันทึกประวัติการทวงถามลงตาราง AuditLogs
  • ชุดรายงานนโยบายแปลง Base64 สู่ผู้บริหาร (AI Executive Briefing Pack): ถ่ายภาพหน้าจอของ ECharts ออกมาเป็น Base64 PNG จากนั้นผสานบทสรุปเชิงลึกของ AI พิมพ์แบบประทับตรา A4 แล้วทำการส่งตรงไปยังอีเมลส่วนตัวของหัวหน้าหน่วยงานในคลิกเดียว

3. ผลการทดสอบการรับมอบจากผู้ใช้ (UAT) และการลงนาม
3.1 ตารางผลการทดสอบการยอมรับระบบโดยผู้ใช้งาน (UAT Test Summary)

การทดสอบ UAT จัดขึ้นระหว่างวันที่ ๑ - ๓ มิถุนายน ๒๕๖๙ โดยมีตัวแทนผู้ใช้จากสพม.สกลนคร และสถาบันการศึกษารวม ๑๕ คนเข้าร่วม มีผลการทดสอบดังนี้:

รหัสทดสอบ หัวข้อทดสอบการใช้งาน เกณฑ์การยอมรับ (Acceptance Criteria) ผลการทดสอบ หมายเหตุ
UAT-01 การลงชื่อเข้าใช้งานและการรักษาเซสชัน สลับสิทธิ์ได้ถูกต้อง และขยายเวลาเซสชันด้วย Heartbeat ทุก 25 นาทีสำเร็จ ผ่าน (100%) เสถียร
UAT-02 การสร้างฟอร์มจัดเก็บข้อมูลเชิงลึก ผู้ดูแลระบบสามารถเพิ่มฟิลด์ประเภทต่างๆ และจำลองฟอร์มออกมาแสดงทันที ผ่าน (100%) ดีเยี่ยม
UAT-03 การบันทึกแบบร่างและส่งข้อมูล สังกัดโรงเรียนบันทึกแบบร่างได้ และเปลี่ยนสถานะเป็นส่งเพื่อรออนุมัติได้ถูกต้อง ผ่าน (100%) ถูกต้อง
UAT-04 การนำเข้าข้อมูลผ่านไฟล์แม่แบบ CSV เปิดไฟล์ CSV ด้วย Excel ภาษาไทยไม่เพี้ยน และตรวจจับข้อผิดพลาดก่อนนำเข้าได้ ผ่าน (100%) BOM ทำงาน
UAT-05 การใช้ AI Conversational BI ส่งคำถามภาษาไทยแล้วได้กราฟแสดงผลบนหน้าจอภายในเวลาเฉลี่ย 3 วินาที ผ่าน (100%) รวดเร็ว
UAT-06 การปักหมุด GIS ค้นหาพิกัด ค้นหาอำเภอ/โรงเรียนผ่าน OSM Nominatim API และอัปเดตพิกัดถูกต้อง ผ่าน (100%) แม่นยำ
UAT-07 ระบบ Dunning Email ทวงหนี้ด้วย AI สั่งพิมพ์จดหมายทวงงานผ่าน AI แนบลิงก์ฟอร์มค้างส่ง และส่งเมลออกจริงได้ ผ่าน (100%) ส่งตรงผู้รับ
